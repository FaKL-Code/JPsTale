# Priston Tale — Asset Loader

A jME3 plugin that decodes Priston Tale asset files, providing the foundation for the map texture customization tool. The supported asset types are:

1. **Fields** — Portals, level restrictions, spawn points, monster types, boss types, etc.
2. **Items** — Equipment attributes, random value ranges, etc.
3. **Monsters** — Attack/defense stats, item drops on death, drop rates, etc.
4. **NPCs** — Dialogue, items for sale, etc.
5. **3D Models** — Meshes, animations, materials, etc.
6. **Textures** — Encrypted BMP and TGA images.
7. **Audio** — WAV and BGM audio files.
8. **Effects**
9. **Sky**
